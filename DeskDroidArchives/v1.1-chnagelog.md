\# DeskDroid Firmware



\## Version 1.1 – Feature Improvements



\### Overview



DeskDroid v1.1 improves the usability and reliability of the initial release by refining timer behavior, adding visual feedback, and improving encoder interaction. The firmware retains the same hardware architecture while introducing several UI and functional enhancements.



---



\### Key Improvements



\*\*Enhanced Timer System\*\*



\* Timer now stores \*\*remaining time when paused\*\*.

\* Timer uses a precise \*\*end-time calculation\*\* instead of elapsed-time tracking.

\* Added \*\*repeating alarm beeps\*\* when timer finishes until acknowledged.



\*\*Timer Status Display\*\*

Timer screen now shows clear states:



\* `Edit MM:SS` – when adjusting timer

\* `MM:SS Ready` – timer ready to start

\* `Running` – timer active

\* `Paused` – timer paused

\* `00:00 Done` – timer completed



---



\*\*Encoder Acceleration\*\*



\* Fast rotary movement increases adjustment speed.

\* Allows quicker timer setting for larger values.



---



\*\*Long Press Visual Feedback\*\*



\* Holding the encoder button displays a \*\*progress bar\*\* indicating long-press activation.

\* Improves user feedback for reset/start actions.



---



\*\*Improved Stopwatch\*\*



\* Stopwatch now supports \*\*hour display for long durations\*\*.

\* Shows clear states:



&nbsp; \* Ready

&nbsp; \* Running

&nbsp; \* Paused



---



\*\*Improved Alarm Handling\*\*



\* Timer completion triggers \*\*periodic buzzer alerts\*\*.

\* Alarm stops when user interacts with the timer.



---



\*\*UI Refinements\*\*



\* Improved screen messages and state feedback.

\* More consistent display refresh behavior.

\* Slightly adjusted button hold timing for better responsiveness.



---
